1. Basics of WEB and frontend

The front-end of a website is made up of three typesd of files: HTML, CSS and JavaScript. these are the files what is loaded in the browser. and called a website.

Learn HTML

HTML, or HyperText Markup Language, the foundation and building block of all websites. its the main file type that is loaded in the browser. the HTML file contain all the content of the page, and it uses tags to denote different types of content. we, can use html tags to create headings and paragraph, add images hyperlinks and so on. but, HTML tags contains dome basic styling, like a word document.

Learn CSS

CSS, or Cascading Style Sheets, lets you add style and colors to make the boring HTML content to look and fancy and visually appealing, it makes the contents more user friendly and readable. you can add whatever you want to satisfy your desire. you can even create animation and shapes with CSS!

To learn css you must create, Whatever you learn always try to create something with it.

Learn JavaScript

JavaScript is a prgramming language designed to run in the browser, JavaScript can make your website dynamic and inteactive to user.

JavaScript is a really fun language to learn and if you have familarity with any other functional programming language. learning JavaScript is preety easy for you, i strongly suggest you to learn vanilla js and clear your Basics before moving to any framework like angular or react.
To learn about them visit MDN its a very well structured resource for a begineers and you can refer to youTube videos also...
