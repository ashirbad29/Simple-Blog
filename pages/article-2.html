1. Install a text editor

Text editors are those piece of softwares where you will write your code, altough you can write code in notepad also but thats not fun at all. there are bunch of very good options out there, below are some editors i personally like.

Top 3 text editors for WEBD

  1. VS code
  2. Sublime text 3
  3. Atom

Those are some grat text editors to start with, but i prefer vs code and sublime text for most of my coding. i suggest you try out yourself and pick the best that suits you.

2.install nodeJs

Well, if you want to run javaScript on your local machine or in your command line, you need nodeJs it enables you to compile and run javaScript locally without a browser.

3.additional things

After allof this pain, now you can start writing code, before that i suggest you to download some great vs extensions that will make your life easier.

  1. Live server
  2. Css peak
  3. vs code intellisense

i cant write here all things how these extensions are helpful, you have to read their documentation, its a good pratice to read documentations before trying out a new things. but i am sure these extensions will help you immensly... Thats all for this.
